=========================
boto3facade
=========================

A simple facade for boto3 that makes some common tasks easier.


Installation
============

.. code-block:: bash
    pip install git+https://github.com/InnovativeTravel/boto3facade


Development environment
=======================

To setup the development environment::

    make develop
    . .env/bin/activate
